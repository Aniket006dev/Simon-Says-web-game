🧠 Simon Says - How to Play

  • 🔊 Start Game: Press any key to begin the game.
  • ✨ Watch Closely: A box will blink indicating the color in the sequence.
  • 👆 Click: Repeat the sequence by clicking the boxes in the same order.
  • ❌ Wrong Move: If you click the wrong box, the game will end.
  • 🚀 Level Up: A new color is added to the sequence with each correct round.
  • 🔁 Restart: Press any key again to restart after a wrong move.
